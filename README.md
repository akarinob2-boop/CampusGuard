#CampusGuard

校园网络违规内容智能识别系统

CampusGuard 是一个面向高校的网络内容安全 AI 系统，融合 动态词典注入 与 对抗鲁棒训练 技术，实现敏感信息的高精度检测与自动拦截。

系统架构
前端：动态监控界面，支持敏感内容实时展示。
后端：Python 服务，处理文本输入并调用 AI 模型进行识别。
AI 模型：
基于深度学习的文本分类模型
通过动态词典注入增强语义理解
对抗训练提升模型鲁棒性，对抗复杂或规避文本
部署：容器化部署（Docker），支持快速上线和扩展
核心技术
动态词典注入：系统可实时加载和更新敏感词库，减少人工维护压力，并提升模型在特定场景下的适应性。
对抗鲁棒训练：通过对抗样本训练，提升模型对文本变异、拼写扰动及规避策略的识别能力。
全链路工程实现：从前端监控、后端服务到模型部署，形成完整可运行的闭环系统。
安装与运行
# 克隆项目
git clone https://github.com/yourusername/CampusGuard.git

# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py

访问 http://localhost:5000 可查看前端界面并进行内容监控。
